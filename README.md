# project-app-todo-list-cli-bayufirmansyah
sebuah aplikasi sederhana todo list dengan menggunakan cli cobra

# command flags
| Flags |  example  | Decription  |
| :---:   | :---: | :---: |
| add | --add "task value" --priority "priority value" | add task with priority |
|  | --add "task value" | add task with default priority (low) |
| list | --list | show all task |
| update | --update taskId --priority "priority value" --status "status value" | update task priority and status |
| delete | --delete taskId | delete task |
| serach | --search "task name" | find task by name |
